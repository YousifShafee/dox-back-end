
    install docker app on windows from:
        https://docs.docker.com/get-docker/

    open terminal on project directory root

    to run project server:
        run command:
            docker build -t dox_web .
        then run command:
            docker run -p 8000:8000 dox_web
    
    to stop project server:
        run command:
            docker ps           // to get <CONTAINER ID> value of project
        then run command:
            docker stop <CONTAINER ID>

    
    valid url:
        localhost:8000/admin
